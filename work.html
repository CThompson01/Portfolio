Game of Life Game of Life is a program that simulates life using a simple grid system.

Rules:
If a live cell has between 2 or 3 live cells around it, it will stay alive.
If a live cell has less than 2 or more than 3 lives cells around it, it will die.
If a dead cell has exactly 3 live cells around it, it will be birthed.
